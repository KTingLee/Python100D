我是 07.js 的配合範例 a.html

雖然你前往的頁面寫 haha.html 但實際上沒有 haha.html

而 haha.html 事實上是引入了 a.html 的內容

也就是說，Node.js 的網址可以做出偽裝的功能

實際上，還有一個問題，就是

a.html 其實有引用一個樣式表(css)，可以用開發者模式(f12)來查看

進入 Network 中，可以看到 css.css 在其中(如果沒看到請重新整理)

這時如果點進去會發現頁面跳轉成"頁面不存在"

因為 a.html 要引用 css.css 時，其實是存取 localhost:3000/css.css

但 css.css 並沒有在路由清單中，也因此這個請求是無效的

解決辦法，將以下程式碼貼入至 07.js 的 req.url == "/haha.html" 的下一段：

else if(req.url === "/public/css.css"){ fs.readFile("./public/07_a.html", function(err, data){ res.end(data); });
