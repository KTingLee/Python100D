這張圖引用路徑為 "./07_subdir/07_picb.jpg"，上一個範例是"./07_pic.jpg"

若要成功顯示圖片，則 localhost:3000/07_subdir/07_picb.jpg 要能夠看到圖片！

整理一下

07_b.html 引用的圖片路徑為 ./07_pic.jpg，連上 07_b.html 的路由為 localhost:3000/GG.html，所以 圖片的路由為 localhost:3000/07_pic.jpg

07_c.html 引用的圖片路徑為 ./07_subdir/07_picb.jpg，連上 07_c.html 的路由為 localhost:3000/GG2.html，所以 圖片的路由為 localhost:3000/07_subdir/07_picb.jpg
